# Diabetes Health Indicators Analysis

## Project Overview
This project analyzes diabetes health indicators using Python data analysis techniques.

Goal:
- Understand diabetes risk factors
- Explore health patterns
- Build data-driven insights


## Dataset
Diabetes Health Indicators Dataset (BRFSS 2015)

Rows: 253,680  
Columns: 22


## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab


## Project Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. GroupBy & Pivot Analysis
5. Correlation Heatmap
6. Insights & Conclusions


## Key Insights
- Older age groups show higher diabetes prevalence.
- Physical activity reduces diabetes risk.
- BMI strongly correlates with diabetes probability.
